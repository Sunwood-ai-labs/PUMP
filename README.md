## 🌟 概要

PUMP（Personalized Utility for Muscle Progress）は、音声入力で筋トレ記録を簡単に管理し、視覚的にわかりやすいグラフを生成するアシスタントです。ユーザーの筋トレ習慣の定着をサポートし、モチベーションを高めることを目的としています。

## 🎥 デモ

[![PUMP Demo](https://github.com/Sunwood-ai-labs/PUMP/assets/108736814/95d80d89-4cf7-45d7-b533-6413d1b786fb)](https://www.youtube.com/watch?v=CHtjaHlrN64)

## 🚀 主な機能

- 音声入力による筋トレ記録の簡単な管理
- 事前定義された筋トレ種目リストからの選択
- コードインタプリタを使用したカラフルでパステルカラーのグラフ生成
- フレンドリーでモチベーショナルな応答
- ユーザーの筋トレ進捗の称賛とサポート
